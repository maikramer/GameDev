# Plugin `navmesh`

Usa `three-pathfinding` com zona gerada a partir de um plano por omissão (ou entidade `<nav-mesh>`). Agentes `<nav-agent>` calculam caminho em `simulation` e movem `transform` em `fixed`.

**Out-of-scope:** carregar GLB de navmesh em runtime (estrutura preparada para extensão com `GLTFLoader`).
