# Generated models (handoff)

Copy GLB files from your **GameAssets** batch output (or `text3d` / `paint3d` CLI) into this folder.

The example expects:

- `hero.glb` — loaded by [`src/main.ts`](../src/main.ts) at URL `/assets/models/hero.glb`.

You can rename the file and update the path in `main.ts`, or add more `loadGltfToScene` calls.

See the monorepo doc: [`docs/MONOREPO_GAME_PIPELINE.md`](../../../../../docs/MONOREPO_GAME_PIPELINE.md) (repository root).
