# Python 3.13 Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate all Python packages (except Rigging3D) to require Python >=3.13, enabling bpy 5.1.0 across the monorepo.

**Architecture:** Update `requires-python`, dependency version floors, classifiers, CI matrix, and installer infrastructure. CUDA 13.2 on the system means PyTorch installs from PyPI (no --index-url needed). Rigging3D stays on 3.11 due to open3d.

**Tech Stack:** Python 3.13, PyTorch 2.6+, bpy 5.1.0, CUDA 13.2
